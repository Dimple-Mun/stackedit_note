# Application对象
- Excel中的所有对象最终都属于`Application`所有
## 属性
- Cells (r,c)
`Application.Cells (r,c)`或省略`Application.`直接写`Cells (r,c)`调用的是Application对象的Cells属性，是当前活动工作表的Cells;
`Worksheets`对象也有这一属性
- Range
- ActiveCell
- Selection
